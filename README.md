# hydrophone_module
hydrophone

SM111 цилиндр можно припаять напрямую к буферу OPA1642, и это будет работать как гидрофон для записи подводных звуков.
Но это работает ТОЛЬКО для приёма звука (пассивный режим):

SM111 PZT цилиндр (26×22×13 мм)
├─ Верхняя серебряная поверхность ──[припой] ──→ Signal In (pin 3 на буфере)
└─ Нижняя серебряная поверхность ──[припой] ──→ Ground (pin 2 на буфере)

⚠️ Осторожность: максимум 350°C, максимум 3 секунды припаивания
(выше Curie temperature PZT испортится)

✅ Запись подводных звуков

✅ Мониторинг кавитации в ванне (как микрофон)

SM111 (гидрофон): Qm ≈ 400-600

Низкий Q = широкая полоса пропускания = хорошая для слабых сигналов от любого направления

Потери энергии выше = не идеально для резонанса

Чувствителен к давлению (датчик)

PZT-8 (мощность): Qm ≈ 800-1200

Высокий Q = узкая полоса пропускания = максимум мощности на резонансе

Потери энергии минимальны = отлично для резонансного режима

Может выдавать большую мощность (передача)

Где купить SM111
Steminc (официальный источник):

Артикул: SMC26D22H13111

Размер: 26×22×13 мм, 42 кГц

Цена: ~$25-30 за штуку

Другие размеры: Доступны от 15 мм до 50 мм диаметра, частоты от 10 до 150 кГц

Форм-факторы доступные у Steminc:

Цилиндры (22-26 мм внешний диаметр, 10-13 мм высота)

Диски (20-50 мм диаметр, 1-5 мм толщина)

Кольца (для встройки в трансдьюсеры)

Полная DIY Hydrophone сборка (по Instructables)
Спецификация из World-Class Hydrophones tutorial:

Пьезокерамика: SM111 цилиндр 26×22×13 мм (~$20)

Буферная плата: JLI Electronics P48 Hi-Z Piezo Buffer (~$45 собранная)

Чип: OPA1642 (низкий шум, low distortion)

Входное сопротивление: 1 МОм

Питание: 12V от phantom power микрофонного предусилителя

Корпус: Уретановая смола Smooth-On 325 (плотность 1.05 g/cc ≈ вода)

Кабель: Микрофонный кабель Mogami (водостойкий для подводных приложений)

Выход: XLR разъём (стандартный микрофонный)

Результат:

Частотный диапазон: До 100+ кГц (за пределы человеческого слуха)

Динамический диапазон: ~100 дБ

Направленность: Омнидирекциональная (слышит со всех сторон)

Стоимость: ~$100 за пару гидрофонов
